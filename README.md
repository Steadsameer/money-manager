# Money Manager Application

A React.js application that helps users manage their finances by tracking income and expenses.

## Features

* Add Income Transactions
* Add Expense Transactions
* Real-Time Balance Calculation
* Transaction History
* Delete Transactions
* Responsive Design

## Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3
* UUID

## Getting Started

```bash
npm install
npm start
```
